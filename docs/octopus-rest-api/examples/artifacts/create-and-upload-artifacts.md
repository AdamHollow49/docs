This script provides an example of how to programmatically upload an [artifact](/docs/projects/deployment-process/artifacts.md) to a deployment process.

## Usage

Provide values for the following:

- Octopus URL
- Space Name
- API Key
- Project Name 
- Release Version
- Environment Name
- File Path to Upload
