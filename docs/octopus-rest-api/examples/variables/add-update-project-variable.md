This script demonstrates how to programmatically add or update a project variable.

## Usage

Provide values for the following:

- Octopus URL
- Octopus API Key
- Name of the space to use
- Name of the project
- The variable properties including:
  - Variable name
  - Variable value
  - Variable type
  - If the variable is sensitive
